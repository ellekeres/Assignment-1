Some Concepts We've Covered:

  1. Document Object Model (DOM)
    1. Programming interface that treats an HTML or XML document as a tree structure
    2. Each node of the tree is an object which is a representation of a piece of the document
    3. Event handlers are what are attached to nodes which are executed when events are detected

  2. HTML Class vs ID
    1. An ID selector is used to identify one single element
    2. A Class selector can be used to identify more than one element
    3. A Class uses a "." to precede a name and an ID uses a "#" to precede a name

  3. Block vs Inline
    1. Block elements will always break onto a new line
    2. Inline elements will never break onto a new line
    3. Block elements include <div> <main> <header> <p> among others, and Inline elements include <span> <button> <image> among others
